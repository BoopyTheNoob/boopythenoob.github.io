# First Order Differential Equations

{: .highlight}
Find the general solution of the following ordinary differential equation:
$$\left(\dfrac{2x}{y}+e^{y\sin x}y\cos x\right)dx+\left(\dfrac{1-x^2}{y^2}+e^{y\sin x}\sin x\right)dy=0.$$
